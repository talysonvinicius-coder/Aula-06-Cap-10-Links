Explicação do Model

Model (Modelo): Representa a lógica de negócio e os dados do sistema. Esta camada é responsável por armazenar, manipular e gerar os dados que serão exibidos na aplicação, sem ter conhecimento sobre como esses dados serão apresentados ou como a interação do usuário ocorre.

Principais funções do Model

  • Armazena e gerencia dados
  • Executa regas de negócio
  • interage com o banco de dados

Exemplo

Imagine um sstema de usuário onde o Model contem as infromações como nome, email e senha, além de regras de validação.
